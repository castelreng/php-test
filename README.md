## Requirements

-   Docker installed/running

## Setup the project

-   Open a terminal
-   `git clone` the project on your machine
-   `cd laravel-app`
-   Provide a valide token into `VOX_TOKEN` variable in `.env` file

## How to start the project

-   `cd laravel-app`
-   `./vendor/bin/sail up -d`
-   go to `http://localhost/organizers/list`
